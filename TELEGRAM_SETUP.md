# Відправка заявок у Telegram

Заявки з форм відправляються **прямо з сторінки** через Telegram Bot API (без сервера).

## Що зробити

1. **Дізнайся Chat ID** бесіди, куди слати заявки:
   - Додай бота [@sms_ybc_warsaw_bot](https://t.me/sms_ybc_warsaw_bot) у потрібну бесіду (групу/канал).
   - Напиши в бесіді будь-яке повідомлення.
   - Відкрий у браузері:  
     `https://api.telegram.org/bot8540186250:AAHL4Wqn1WU-gfZt_yf7HXCk8qZpLUJciag/getUpdates`
   - У відповіді знайди `"chat":{"id": -123456789` — це і є Chat ID (число, для груп з мінусом).

2. **Підстав Chat ID у `index.html`**  
   Знайди рядок:
   ```js
   const TELEGRAM_CHAT_ID = ''; // ID бесіди
   ```
   Замість порожніх лапок вкажи ID, наприклад:
   ```js
   const TELEGRAM_CHAT_ID = '-1001234567890';
   ```

Збережи, закоміть і запуш — після цього заявки будуть приходити в обрану бесіду.

---

**Примітка:** Токен бота і Chat ID зберігаються в коді сторінки, їх можна побачити в DevTools. Якщо потрібна повна конфіденційність — можна повернутися до варіанту з серверною функцією (Vercel) і винести токен у змінні середовища.
